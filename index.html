Un exemple de photomontage

  1. Lien vers l'image 1
  2. Lien vers l'image 2
  3. Procédure :
    1. Ouvrir les images dans Photoshop ;
    2. Choisir l'outil de sélection "lasso magnétique" ;
    3. Détourer sur l'image 2 la portion d'image que je veux copier avec le lasso magnétique" ;
    4. Utiliser CTRL+C pour copier la sélection ;
    5. Changer d'image et passer sur l'image 1 ;
    6. Utiliser CTRL+V pour coller la sélection précédente ;
    7. Enregistrer l'image au format .jpg.
  4. Lien vers le photomontage
